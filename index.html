You-torial

You-Torial is a responsive, full CRUD app serving as a collection of online development tutorials. Users provide ratings/reviews to lessen time-consuming searches.

Technologies used: React.JS, Node.js, Express

GeoTube

GeoTube is a web app where the primary function is to find YouTube videos based on locations inputted by the user. GeoTube makes use of Google and YouTube’s API’s to display YouTube videos relevant to specific locations.

Technologies used: React.JS, Ruby on Rails, PostgreSQL

Streamr

Streamr is a single page web app that is designed to display various information about movies chosen by the user. Streamr uses JQuery to make Ajax calls to three API’s to get necessary data.

Technologies used: JavaScript, JQuery

Bingo

Bingo is a two player, online version of the classic and fan favorite game.

Technologies used: JavaScript, HTML/CSS
